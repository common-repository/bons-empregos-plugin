== Description ==

PT: O plugin Bons Empregos adiciona um widget ao teu blog que mostra as ultimas ofertas de [emprego](http://bonsempregos.com "Ofertas de emprego.") e pode ser integrado em qualquer sitio do teu blog. 


EN: The Bons Empregos Plugin adds a customizable widget which displays the latest [job offers](http://bonsempregos.com "Job offers.") from Portugal. It can be integrated anywhere in the blog. This Jobticker shows up the last five or more job offers and is a great solution to help spread job opportunities.

== Installation ==

*Could not be simpler*
1. Extract `bons-empregos-plugin.zip`.
2. Upload the `bons-empregos-plugin`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the Plugins menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
